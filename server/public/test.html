AutoPartsDV — Тест API

Health Check

GET /api/health

Регистрация

POST /api/auth/register — создаёт пользователя и возвращает токен

Вход

POST /api/auth/login — возвращает токен

Профиль

GET /api/profile — требуется токен

PUT /api/profile — обновить имя и/или email

Поиск запчастей

GET /api/search?q=... — доступен всем, с токеном сохраняет историю

Запчасть по артикулу

GET /api/parts/:article

История поиска

Требуется токен

DELETE /api/history/:id — удалить одну запись

Поставщики

GET /api/suppliers — доступен всем

POST /api/suppliers — создать (только admin)

PUT /api/suppliers/:id — обновить статус (только admin)

DELETE /api/suppliers/:id — удалить (только admin)
